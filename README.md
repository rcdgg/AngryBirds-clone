# Angry Birds Clone

---

## How to run :
 -  Ideally open the project in Intellij Idea(other editors work too).
 - Go to lwjgl3/src/main/java/birds/angry/lwjgl3/Lwjgl3Launcher.java and run the main function there to launch the app
